Modul 3: Backend dengan Node.js

Bagian dari Fullstack Web Developer Path

Tentang Pelajaran Ini

Selamat datang di modul backend! Di pelajaran ini, kita akan memulai perjalanan kita dengan Node.js, sebuah runtime JavaScript yang memungkinkan kita menjalankan JavaScript di sisi server. Kita akan membahas:

  • Apa itu Node.js dan mengapa ia populer.
  • Setup environment Node.js di komputer Anda.
  • Membuat server HTTP pertama Anda.
  • Memahami konsep dasar asynchronous di Node.js.

Pastikan untuk menginstal Node.js versi LTS terbaru sebelum melanjutkan. Anda dapat mengunduhnya dari situs resmi Node.js.
